Crear un REST Full API con Spring Boot 3 con los siguientes endpoints:

POST /api/v1/createEmployee
GET /api/v1/getEmployees
GET /api/v1/getEmployee
PUT /api/v1/updateEmployee
DELETE /api/v1/deleteEmployee

POST /api/v1/createCompany
GET /api/v1/getCompanies
GET /api/v1/getCompany
PUT /api/v1/updateCompany
DELETE /api/v1/deleteCompany

Los datos de la entidad de empleados debe tener:
id, nombres, apellidos, fecha nacimiento

Los datos de la entidad de compañía debe tener:
id. nombre, dirección
